Basic GET:

```bash
curl -sS https://example.com
```

Download to a file:

```bash
curl -sS -o page.html https://example.com
```

## Tip: `-I` fetches headers, `-L` follows redirects.

Fetch a URL and show response headers:

```bash
curl -I https://example.com
```
