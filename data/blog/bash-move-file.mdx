Rename or move a file:

```bash
mv oldname.txt newname.txt
mv file.txt /path/to/dest/
```

## Tip: use `mv -i` to prompt before overwrite.

Use mv to move or rename files.

```bash
mv a.txt b.txt
# move into dir
mv file.txt dest_dir/
```

Tip: `mv -n` avoids clobbering; `mv -b` creates backups on overwrite.
