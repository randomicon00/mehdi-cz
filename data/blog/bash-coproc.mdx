summary: "Use coproc to run a process and communicate with it via file descriptors."
---

Start a coprocess:

```bash
coproc myproc { some-server --interactive; }
echo "request" >&${myproc[1]}
read -r response <&${myproc[0]}
```

Tip: coproc is handy for talking to interactive commands without ptys.
