---
title: "Source a script into the current shell"
date: "2025-09-16"
tags: ["bash", "til"]
draft: false
summary: "Use `source` or `.` to run a script in the current shell so it can set variables/functions."
---

```bash
source ./env.sh
. ./env.sh
```

Warning: Sourcing runs code in your shell; review scripts before sourcing.
