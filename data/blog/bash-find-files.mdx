Find files by name:

```bash
find . -name "*.log"
```

Find and delete empty files:

```bash
find . -type f -empty -delete
```

## Tip: combine with `-exec` or `-print0 | xargs -0` for safe handling of spaces.

Find files named 'config.json' from current dir:

```bash
find . -type f -name 'config.json'
```
