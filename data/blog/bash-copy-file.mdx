Copy a file:

```bash
cp src.txt dest.txt
```

Copy recursively:

```bash
cp -r srcdir/ destdir/
```

## Tip: `cp -a` preserves attributes and is useful for backups.

tags: ["bash","til"]

summary: "Use cp to copy files and preserve attributes when needed."

---

```bash
cp source.txt dest.txt
# preserve mode, ownership and timestamps
cp -a source.txt dest_dir/
```

Tips:

- Use `cp -r` for directories.
- `cp -a` keeps metadata; use sudo when copying system files.
