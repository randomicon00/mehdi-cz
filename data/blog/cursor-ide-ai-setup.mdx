Cursor IDE is built for AI-first development. Learn to leverage its unique features for faster, smarter coding.

## Key Features Setup

**Composer (Ctrl+I):** Multi-file AI editing

```
Enable in settings: AI > Composer > Auto-apply simple changes
```

**Chat (Ctrl+L):** Contextual code discussions

```
@workspace - Include entire codebase context
@file - Reference specific files
@code - Discuss selected code
```

**Tab Autocomplete:** Intelligent code completion

```
Settings > AI > Tab Autocomplete > Enable
Model: GPT-4 (for better accuracy)
```

## Effective Workflows

**Refactoring:**

```
1. Select code block
2. Ctrl+I: "Extract this to a reusable hook"
3. Review and apply changes
```

**Bug fixing:**

```
1. Ctrl+L with error context
2. Ask: "This function throws [error]. Fix it and explain why"
```

**Code review:**

```
1. Select function/class
2. Ctrl+L: "Review this code for performance and security issues"
```
